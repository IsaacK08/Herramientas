COMANDOS: mkdir (crear carpeta nueva) / mkdir -v (crear carpeta nueva y el -v sirve para darnos más información acerca de lo ocurrido)
ls: (ver donde estamos parados) / ls -a (ver archivos ocultos)
cd: (para movernos entre carpetas) / cd .. (volver una carpeta atrás)
pwd: (imprimir el directorio)
clear: (limpiar la terminal)
touch: (crea un archivo vacío) es importante definir su extensión.
git rm (elimina archivo seleccionado) / rm -r (elimina carpetas)
help (lista de comandos básicos) / comando x -- help (para obtener ayuda sobre ese comando x)
git add (agregar archivos al siguiente commit) / git add . (agregar todos los archivos que están dentro de la carpeta seleccionada)
git commit -m (agregar el commit junto con un mensaje descriptivo)
git log (muestra los commits que tenemos actualmente)
git branch -M (modifica el nombre de la rama de trabajo ej: main, master etc..)
git remote add origin +link (hace una conexión remota desde nuestro repositorio local con la cuenta github)
git push (guarda camnbios del repositorio local y los envia al repositorio remoto que está en github) / git push -u origin (sincroniza el repositorio local con el repositorio remoto)
git track (rastrea un archivo y le hace un seguimiento)
git status (ver el estado en que se encuentra x archivo)
stage (espacio reservado para agrupar todos aquellos archivos que sumaremos a un proximo commit)
git pull (traer información actualizada del repositorio)
